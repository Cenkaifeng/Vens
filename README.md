# Vens
Taro3.x miniapp 

### 基础描述

Taro3.4 + Vue3.x + TypeScript + Sass + NutUI

##### 宿主环境要求
* MacOS Nodejs ^v14.17.0 Win Nodejs ^14.0.0
* 全局 Tarojs/cli

### 工程初始化

1. 确保全局按照了 `@tarojs/cli` yarn add @tarojs/cli global 要求^v3.4.0
2. 整体要求使用 
    * `yarn `
    * `npm install` 



### Script

1. 开发热更新
`yarn dev:weapp`

2. 开发构建
`yarn build:weapp`
